## Description

---

## Technical Overview

A USB-powered Zigbee mmWave presence sensor that detects fine motion and human presence using radar technology — without relying on traditional PIR sensing.

Enables accurate presence-based automations such as lighting control and HVAC triggers even when minimal movement occurs. USB power allows flexible placement where mains wiring is unavailable.

Key Specifications
- Protocol: Zigbee 3.0
- Detection: mmWave radar (motion + presence)
- Power: USB
- Mounting: Flexible placement

⚠ Does not control electrical loads. Provides presence/motion data for automation triggers.
