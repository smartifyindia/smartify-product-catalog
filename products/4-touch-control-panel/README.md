## Technical Overview

A 4-inch wall-mounted smart home touch panel with an integrated Zigbee gateway, Wi-Fi connectivity, and built-in Alexa voice control. Provides centralized control of smart home devices, scenes, and automations.

Features automatic screen brightness adjustment and supports remote app-based control alongside local touchscreen interaction.

Key Specifications
- Display: 4-inch touchscreen
- OS: Embedded Linux
- Wireless: Wi-Fi + Zigbee (built-in gateway)
- Voice: Built-in Alexa
- Functions: Scenes, devices, automations, status
- Installation: Wall-mounted

⚠ Does not switch electrical loads. Control interface and Zigbee coordinator only.
