## Technical Overview

An 8-button Zigbee capacitive touch panel with an integrated 6A utility socket, combining comprehensive load control and scene triggering with convenient power access in a single wall panel.

Each button is independently configurable for toggle or momentary operation with dimmer support via retrofit modules. Available in white or black glass with silver, gold, or black bezel.

Key Specifications
- Buttons: 8 capacitive touch
- Socket: 1 integrated (6A)
- Relay: 10A per button channel
- Direct Load: Up to 4A per button channel
- Protocol: Zigbee 3.0
- Installation: Wall-mounted

⚠ For loads above 4A, use an external SSR or contactor. Socket rated at 6A.
