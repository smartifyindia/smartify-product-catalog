## Description

A room that knows when it's occupied.

The Ceiling mmWave Sensor watches the whole room — not a narrow cone. It detects presence, not just motion, so every automation responds to whether someone is actually there. Not whether they waved.

---

## Technical Overview

A ceiling-mounted, mains-powered Zigbee mmWave presence sensor designed for accurate occupancy detection across a full room. Detects micro-movements for continuous presence sensing — not just initial motion.

Ideal for lighting control, HVAC triggers, and occupancy-based automation in residential and commercial spaces. Ceiling mount provides optimal coverage.

Key Specifications
- Protocol: Zigbee 3.0
- Detection: mmWave radar (motion + presence)
- Power: Mains powered
- Mounting: Ceiling mount

⚠ Does not control electrical loads. Provides presence/motion data for automation triggers.
