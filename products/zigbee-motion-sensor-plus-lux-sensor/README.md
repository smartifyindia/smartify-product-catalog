## Technical Overview

A Zigbee 3.0 combined motion and ambient light sensor that reports both occupancy and LUX readings to a Zigbee gateway or compatible hub for automation and control logic.

Enables intelligent automations based on both presence and light conditions — for example, activating lights only when motion is detected and ambient light is below a set threshold. Battery-powered with wall or ceiling mount.

Key Specifications
- Protocol: Zigbee 3.0
- Detection: PIR motion + ambient light (LUX)
- Power: Battery
- Mounting: Wall or ceiling

⚠ Does not control electrical loads. Provides sensor data for automation triggers.
