## Technical Overview

A 10-inch wall-mounted smart home control panel with integrated Zigbee and Bluetooth gateways, supporting 128+ devices per protocol. Features a built-in HD camera, doorbell intercom, proximity wake sensor, and customizable control interface.

Supports Wi-Fi, Zigbee, and Bluetooth. Powerable via live/neutral wiring or USB-C. Manages lighting, scenes, security, and automation from a central touchscreen.

Key Specifications
- Display: 10-inch touchscreen

- Capacity: 128+ Zigbee + 128+ Bluetooth
- Camera: HD ~5MP (intercom + monitoring)
- Features: Doorbell intercom, proximity sensor
- Power: Live/neutral or USB-C
