---

## Technical Overview

A Zigbee 3.0 magnetic contact sensor that detects and reports the open/closed state of doors, windows, and other entry points to a Zigbee gateway or compatible hub.

Battery-powered with surface mount installation via adhesive or screws. Used for security monitoring and as an automation trigger.

Key Specifications
- Protocol: Zigbee 3.0
- Detection: Magnetic open/close
- Power: Battery
- Mounting: Surface mount (adhesive or screws)

⚠ Status sensor only — does not control electrical loads.
