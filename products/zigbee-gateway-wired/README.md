## Technical Overview

A wired Zigbee 3.0 gateway that acts as the central coordinator for all Zigbee devices in the system. Connects via Ethernet for a stable, low-latency network backbone.

Supports up to 128 Zigbee devices and enables device pairing, scene automation, remote control, and unified device management through compatible mobile applications and automation platforms.

Key Specifications
- Device Type: Zigbee Coordinator / Gateway
- Protocol: Zigbee 3.0
- Network: Wired Ethernet (LAN)
- Capacity: Up to 128 Zigbee devices
- Installation: Indoor, fixed LAN connection

⚠ Does not control electrical loads. A stable wired Ethernet connection is required for reliable operation.
