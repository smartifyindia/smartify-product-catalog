## Technical Overview

Convergia is a locally hosted smart automation gateway that unifies KNX wired systems and wireless ecosystems — Zigbee, Wi-Fi, Matter, and LAN/IP — into a single interoperable platform.

Built on an Intel NUC-based appliance, it runs all automation logic locally, ensuring reliability independent of cloud connectivity. Advanced rule-based automation supports complex triggers, conditions, and multi-step logic. Integrates AV systems and IP cameras directly into automation workflows.

Key Specifications
- Architecture: On-premise (locally hosted)
- Protocols: KNX, Zigbee, Wi-Fi, Matter, LAN/IP
- AV Integration: Yes
- Camera Integration: IP cameras
- Automation: Rule-based, local execution
- Interface: Web dashboard + mobile access
- Hardware: Intel NUC-based pre-configured appliance

⚠ Does not switch electrical loads. Functions as the central intelligence layer for the entire automation system. Qualified integrator installation recommended.
