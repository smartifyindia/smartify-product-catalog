## Technical Overview

A single-channel Zigbee phase-cut dimmer module designed to retrofit any standard wall switch with intelligent brightness control — no switch replacement required.

Supports trailing and leading edge phase-cut dimming up to 200W for compatible dimmable lighting fixtures and drivers. Installs behind the existing switch in a flush box or on DIN rail. Communicates natively over Zigbee 3.0 with Home Assistant, and integrates with Matter and Apple HomeKit via a compatible hub.

Key Specifications
- Channels: 1
- Max Load: 200W
- Dimming Type: Phase-Cut (Trailing/Leading edge)
- Protocol: Zigbee 3.0
- Compatible with: Home Assistant (native), Matter & Apple HomeKit (via hub)
- Installation: Flush box or DIN rail

⚠ For use with dimmable fixtures and drivers only. Not a 0-10V controller and cannot function as a plain on/off relay.
