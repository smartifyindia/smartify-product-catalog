## Technical Overview

A Zigbee smart fan control panel for ceiling fans up to 150W. Provides on/off and speed control with a dedicated fan control interface.

Integrates with the Zigbee automation ecosystem. Available in PP or anti-glare glass in black, white, or gold.

Key Specifications
- Control: Fan on/off + speed
- Fan Load: Up to 150W
- Protocol: Zigbee 3.0
- Materials: PP or AG glass
- Colours: Black, White, Gold
- Installation: Wall-mounted

⚠ For ceiling fans within rated load (150W) only.
