## Technical Overview

A dual-channel Zigbee phase-cut dimmer that retrofits two standard wall switches with independent brightness control — without replacing any existing hardware.

Each channel independently supports phase-cut dimming up to 100W for compatible dimmable lighting fixtures and drivers. Installs behind existing switches in a flush box or on DIN rail. Communicates natively over Zigbee 3.0 with Home Assistant, and integrates with Matter and Apple HomeKit via a compatible hub.

Key Specifications
- Channels: 2 (independent)
- Max Load: 100W per channel
- Dimming Type: Phase-Cut (Trailing/Leading edge)
- Protocol: Zigbee 3.0
- Compatible with: Home Assistant (native), Matter & Apple HomeKit (via hub)
- Installation: Flush box or DIN rail

⚠ For use with dimmable fixtures and drivers only. Not a 0-10V controller and cannot function as a plain on/off relay.
